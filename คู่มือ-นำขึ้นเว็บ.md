# 🌐 คู่มือนำ Investing Pro ขึ้นเว็บ (Render.com — ฟรี)

ทำตาม 3 ขั้นตอนนี้ จะได้ลิงก์ถาวรส่งให้เพื่อนใช้ได้เลย เช่น
`https://investing-pro.onrender.com`

---

## ขั้นที่ 1: สมัคร GitHub และสร้าง repo (~10 นาที)

1. เข้า **https://github.com/signup** สมัครด้วยอีเมล (ฟรี)
2. กดปุ่ม **+** มุมขวาบน → **New repository**
3. ตั้งชื่อ `investing-pro` → เลือก **Private** ก็ได้ → กด **Create repository**
4. จะเห็นหน้าที่มี URL ประมาณ `https://github.com/ชื่อคุณ/investing-pro.git` — **ก๊อป URL นี้ไว้**

## ขั้นที่ 2: อัปโหลดโค้ด

เปิด PowerShell แล้วรันทีละบรรทัด (แทน `<URL>` ด้วยที่ก๊อปมา):

```powershell
cd "C:\Users\USER\Desktop\For claude\investing-pro"
git remote add origin <URL>
git push -u origin main
```

(ครั้งแรก git จะเด้งหน้าต่างให้ล็อกอิน GitHub — ล็อกอินผ่านเบราว์เซอร์ได้เลย)

> 💡 หรือบอก Claude ว่า "push ขึ้น GitHub ให้หน่อย URL คือ ..." เดี๋ยวจัดการให้

## ขั้นที่ 3: Deploy บน Render (~10 นาที)

1. เข้า **https://render.com** → **Get Started** → เลือก **Sign in with GitHub**
2. กด **New +** → **Web Service**
3. เลือก repo `investing-pro` (ถ้าไม่เห็น กด Configure account ให้สิทธิ์เข้าถึง repo)
4. Render จะอ่านไฟล์ `render.yaml` ที่เตรียมไว้ให้แล้ว — ค่าทุกอย่างถูกตั้งอัตโนมัติ
   (ถ้าถามแผน เลือก **Free**)
5. กด **Create Web Service** รอ build ~3-5 นาที
6. เสร็จแล้วจะได้ลิงก์ `https://investing-pro-xxxx.onrender.com` — **ส่งลิงก์นี้ให้เพื่อนได้เลย!**

## ⚠️ ข้อจำกัดของแผนฟรี Render

- **เซิร์ฟเวอร์หลับเมื่อไม่มีคนใช้ 15 นาที** — คนแรกที่เปิดจะรอ ~1 นาทีให้ตื่น จากนั้นเร็วปกติ
- **watchlist ที่แก้บนเว็บจะรีเซ็ต**เมื่อเซิร์ฟเวอร์ restart (ดิสก์ชั่วคราว) —
  ถ้าอยากเปลี่ยน watchlist ถาวร ให้แก้ `config.json` ในเครื่องแล้ว push ขึ้นไปใหม่
- การแจ้งเตือนแบบเด้งหน้าจอ (Windows Toast) ใช้ได้เฉพาะตอนรันบนเครื่องคุณ —
  บนเว็บให้ใช้ Telegram แทน (ใส่ token ใน config.json)
- ทุกคนที่มีลิงก์เห็น watchlist เดียวกันและแก้ได้ — อย่าแชร์ลิงก์ในที่สาธารณะ

## การอัปเดตเวอร์ชันหลังจากนี้

แก้โค้ดในเครื่องเสร็จ → `git add -A; git commit -m "update"; git push` →
Render จะ build ใหม่ให้อัตโนมัติใน ~3 นาที

## 🤖 เปิดใช้ AI วิเคราะห์หุ้น (ฟรีด้วย Google Gemini)

ปุ่ม "🤖 AI วิเคราะห์เจาะลึก" ในการ์ดหุ้นใช้ได้เมื่อใส่คีย์ AI — แนะนำ **Gemini เพราะฟรี**

**ขั้นตอน (ครั้งเดียว ~3 นาที):**
1. เข้า **https://aistudio.google.com/apikey** ล็อกอินด้วย Google → กด **Create API key** (ฟรี ไม่ต้องผูกบัตร)
2. ก๊อปคีย์ที่ได้ (ขึ้นต้น `AIza...`)
3. **บนเครื่องคุณ** — เปิด PowerShell รันบรรทัดนี้ (แทนคีย์) แล้วเปิดโปรแกรมใหม่:
   ```powershell
   [Environment]::SetEnvironmentVariable("GEMINI_API_KEY", "AIzaใส่คีย์ของคุณ", "User")
   ```
4. **บนเว็บ Render** — Dashboard → บริการ investing-proped → **Environment** → Add: `GEMINI_API_KEY` = คีย์ → Save (deploy ใหม่อัตโนมัติ)

**ทางเลือก — ใช้ Claude แทน (คุณภาพสูงกว่าเล็กน้อย แต่จ่ายตามใช้):** ตั้ง `ANTHROPIC_API_KEY` แทน (สมัครที่ console.anthropic.com) ถ้าใส่ทั้งสองคีย์ ระบบเลือก Gemini ก่อน (ฟรี)

> โควตาฟรีของ Gemini กว้างมาก (หลักพันครั้ง/วัน) เพียงพอสำหรับการใช้ส่วนตัวและแชร์เพื่อนกลุ่มเล็ก
