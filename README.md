# 📈 Investing Pro — ระบบแจ้งเตือน & วิเคราะห์หุ้นเชิงเทคนิค

วิเคราะห์หุ้นด้วยอินดิเคเตอร์มาตรฐานสากล ดึงข้อมูลจริงจาก **Yahoo Finance**
รองรับทั้งหุ้นสหรัฐ (NVDA, AAPL, …) และหุ้นไทย (ใส่ `.BK` ต่อท้าย เช่น `PTT.BK`)

## 🖥 วิธีเปิดหน้าจอ Dashboard (แนะนำ)

**ดับเบิลคลิกไฟล์ `เปิดโปรแกรม.bat`** — เบราว์เซอร์จะเปิดหน้า Dashboard ให้อัตโนมัติที่ http://127.0.0.1:8750

Dashboard มี 2 หน้า สลับด้วยแท็บด้านบน:
- **📊 หุ้น** — วิเคราะห์เทคนิค + ความคุ้มค่า + คำแนะนำควรซื้อไหม
- **📰 ข่าว** — ข่าวการเงินล่าสุดจาก CNBC, MarketWatch, WSJ, Reuters ฯลฯ
  รวมข่าวเฉพาะหุ้นใน watchlist **แปลไทยอัตโนมัติทุกข่าว** รีเฟรชเองทุก 5 นาที
  พร้อมลิงก์อ่านต้นฉบับ

บนหน้าจอทำได้ทุกอย่าง:
- 🔄 **สแกนทั้งหมด** — วิเคราะห์หุ้นทุกตัวใน watchlist ใหม่ทันที
- ＋ **เพิ่มหุ้น** — พิมพ์ชื่อในช่องแล้วกด Enter (ระบบตรวจให้ว่าชื่อถูกต้องดึงข้อมูลได้จริง)
- ✕ **ลบหุ้น** — กดปุ่ม ✕ มุมขวาบนของการ์ด
- การ์ดเรียงตามคะแนนอัตโนมัติ ตัวที่น่าซื้อสุดอยู่บนสุด พร้อมแถบสรุป ซื้อ/ถือ/ขาย

## วิธีใช้งานแบบ Command Line

เปิด PowerShell แล้วรัน:

```powershell
cd "C:\Users\USER\Desktop\For claude\investing-pro"

# สแกนหุ้นทั้ง watchlist 1 รอบ + แจ้งเตือน
python investing_pro.py

# วิเคราะห์เจาะลึกหุ้นตัวเดียว
python investing_pro.py --ticker NVDA
python investing_pro.py --ticker PTT.BK

# โหมดเฝ้าระวัง: รันวนทุก 30 นาที แจ้งเตือนเมื่อเกิดสัญญาณใหม่
python investing_pro.py --watch

# ดูรายงานอย่างเดียว ไม่ต้องแจ้งเตือน
python investing_pro.py --no-alert
```

## สิ่งที่วิเคราะห์ให้

| หมวด | รายละเอียด |
|---|---|
| แนวโน้ม | EMA 20 / 50 / 200, Golden Cross / Death Cross |
| โมเมนตัม | RSI(14) เขต oversold/overbought, MACD cross |
| ราคา | Breakout/Breakdown กรอบ 20 วัน, Bollinger Bands |
| วอลุ่ม | Volume spike เทียบค่าเฉลี่ย 20 วัน |
| แนวรับ-แนวต้าน | คำนวณจาก swing high/low จริง 60 วันล่าสุด |
| จุดบริหารความเสี่ยง | จุดตัดขาดทุน (2×ATR) และเป้าหมาย (3×ATR) |

ทุกสัญญาณถูกให้น้ำหนักแล้วรวมเป็น **score** — สัญญาณหลายตัวชี้ทางเดียวกัน (confluence)
ความเชื่อมั่นยิ่งสูง สรุปเป็น: `STRONG BUY / BUY / HOLD / SELL / STRONG SELL`

## การแจ้งเตือน

- **Windows Toast** — เด้งมุมจอทันทีเมื่อเกิดสัญญาณสำคัญ (เปิดใช้อยู่แล้ว)
- **Telegram** (ทางเลือก) — ใส่ `telegram_bot_token` และ `telegram_chat_id` ใน `config.json`
  (สร้างบอทฟรีผ่าน @BotFather ใน Telegram)
- แจ้งเตือนเหตุการณ์เดิม **ไม่ซ้ำในวันเดียวกัน** (จำสถานะไว้ใน `alert_state.json`)

## ปรับแต่งใน config.json

- `watchlist` — เพิ่ม/ลบหุ้นได้ตามต้องการ
- `rsi_oversold` / `rsi_overbought` — เกณฑ์ RSI (ค่าเริ่มต้น 30/70)
- `volume_spike_factor` — กี่เท่าของวอลุ่มเฉลี่ยถึงนับว่าผิดปกติ (2.0)
- `watch_interval_minutes` — รอบสแกนในโหมด `--watch` (30 นาที)
- `min_alert_score` — คะแนนขั้นต่ำที่ trigger การแจ้งเตือนสรุป (3)
